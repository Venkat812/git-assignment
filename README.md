# Travel Recommendation Website

A simple travel recommendation website built using:
- HTML
- CSS
- JavaScript
- Fetch API

## Features
- Home page with background image
- Navigation bar
- Keyword-based search (beach, temple, country)
- Recommendation results using JSON
- About Us and Contact Us pages

## How to Run
1. Clone the repository
2. Open `travel_recommendation.html` in a browser
